# 競プロ説明書

Atcoderを頑張る．
## 基本の流れ
1. **準備**

開催Noに対応するフォルダが作れる
```
acc new abc350
```
2. **フォルダへ移動**
```
cd abc350/a
```
3. テスト実行
```
touch main.py
oj t -c "python3 main.py" -d tests
```
## 便利コマンド
1. 3番目のテスト実行をしてくれる
```
ojt
```
> **注意**
> `ojt` が動かない時は `source ~/.zshrc` を試すこと！
